Desafío Ridículo

Aplicación móvil desarrollada en **Android Studio (Kotlin + Room + MVVM)** que permite gestionar y ejecutar desafíos divertidos y ridículos.  
El proyecto busca fomentar la creatividad, el humor y la interacción social mediante retos simples y personalizables.

---

Características principales
- Lista de desafíos con estados: **Pendiente, En curso, Completado**.
- Creación, edición y eliminación de desafíos.
- Persistencia local con **Room (SQLite)**.
- Arquitectura **MVVM** con Repository.
- Pantalla de administración para moderar desafíos.
- Semilla inicial de desafíos ridículos precargados.

---

Tecnologías utilizadas
- **Lenguaje:** Kotlin
- **IDE:** Android Studio
- **Base de datos:** Room (SQLite)
- **UI:** RecyclerView + ViewBinding + Material Design
- **Arquitectura:** MVVM
